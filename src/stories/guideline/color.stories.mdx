# Color Guideline

우노뷰티 CRM Color Guide는 **Unoblue(#2C62F6)** 을 기본으로 사용하며 표준화된 컬러 및 사용가이드를 제공하여 디자이너가 일관성 있는 브랜드 경험을 만들어 나갈 수 있도록 돕습니다.
디자이너는 목적성 있는 컬러 사용을 통해 유저와 서비스간의 명확한 의사소통이 이루어질 수 있도록 해야합니다.
