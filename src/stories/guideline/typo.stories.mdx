# Typography Guideline

우노뷰티 CRM Typography는 **Noto Sans KR**을 기본으로 사용하며 디자인과 개발단의 효율적인 작업을 가능하게 합니다.  
정의된 타입 스케일과 컬러를 고려하여 적용된 타이포그래피는 콘텐츠의 중요한 정도를 구분하고, 텍스트 전체의 밸런스를 맞추어 사용자의 프로덕트 경험을 향상시킵니다.
