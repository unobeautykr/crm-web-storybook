# Button Guideline

우노뷰티 CRM 에서 유저의 특정 액션을 유도하는 Button 컴포넌트 입니다.  
**Radius 는 2px, 또는 4px**을 사용합니다.  
텍스트 좌우 **margin은 최소 2px에서 Button 크기와 사용 맥락에 맞춰 4의 배수(4, 8, 12, 16, 24 ...)**로 유연하게 사용합니다.
